## Visão Geral

O SyntaxWear é um ecommerce fictício para venda de tênis e sneakers, com foco em experiência do usuário, acessibilidade e boas práticas de HTML, CSS e organização de código.

## Funcionalidades

- Layout responsivo para desktop e mobile
- Menu de navegação central e menu de ícones à direita
- Seção hero com destaque de produto
- Seção de categorias de tênis
- Grid de produtos em destaque
- Footer completo com newsletter, redes sociais e links institucionais
- Uso de CSS modularizado por componentes
- Reset CSS baseado em Andy Bell para consistência cross-browser

---

## Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (com uso de rem para espaçamentos e variáveis)
- Organização modular de CSS por componentes
- SVG para ícones
- Imagens otimizadas para web

---

## Como Rodar o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/JeanKarlo-Dev/ecommerce-syntaxwear.git
   ```
2. Abra o arquivo `index.html` em seu navegador.

> Não é necessário backend ou instalação de dependências para visualização do layout.
